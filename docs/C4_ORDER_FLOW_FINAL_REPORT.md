## 3. C4 Strategy Engine — COMPLETE

Implemented in `scripts/c4_strategy.py`:

### C4 Scoring Formula (0-100)

| Component | Range | Factors |
|-----------|-------|---------|
| C1 Context | 0-25 | BOS/CHOCH strength + OB quality + sweep zone |
| C2 Confirmation | 0-25 | OB retest + rejection candle + close confirms |
| C3 Commitment | 0-25 | Delta + CVD + volume + imbalance + absorption |
| C4 Control | 0-25 | Risk:Reward + SL quality + risk gate pass |

### Decision Thresholds

- **≥80:** PAPER TRADE (auto-execute in paper mode)
- **65-79:** WATCHLIST (monitor only)
- **<65:** REJECT

### Classes

- `C4StructureAnalyzer` — detect_swings, detect_bos_choch, detect_order_blocks, detect_liquidity_sweeps
- `C4Scorer` — evaluate() returns full C1-C4 breakdown
- `C4Backtester` — run() executes historical backtest with trade tracking

## 10. OHLCV Proxy Transparency

### What We Use

```
CLV = ((Close - Low) - (High - Close)) / (High - Low)
Delta_proxy = Volume * CLV
```

### What We DON'T Have

- Real Level 2 / order book data
- Real bid/ask volume split
- Real footprint charts
- Real DOM depth
- Real iceberg detection
- Real tick-by-tick data

### Accuracy Estimate

- Delta proxy: ~60-70% correlation with real delta
- Good enough for directional bias, not for scalping
- All UI and plots clearly labeled "OHLCV PROXY ORDER FLOW"
